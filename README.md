# Multi-Model-Al-system-for-Industrial-Quality-Assurance
An AI-driven computer vision system that automates defect detection on manufacturing lines, ensuring consistent product quality and reducing the need for manual inspection.
Progress Log
Week 1: Python Basics
I kicked things off by setting up my workspace in Google Colab and Kaggle. I spent time getting comfortable with Python basics—like loops, dictionaries, and functions—before touching briefly on the basic theory behind machine learning.

Week 2: Machine Learning Basics
This week was all about classical machine learning. I learned data manipulation using Pandas and studied core ML concepts up through ensemble methods like Bagging, building a solid intuition for how models are actually trained and evaluated.

Week 3: Deep Learning & CNNs
I finally dived into Deep Learning, following a structured video series. The main focus was understanding Convolutional Neural Networks (CNNs) and exactly why they are the go-to choice for image-based tasks, which is crucial for our visual inspection goals.

Week 4: Intro to LLMs
I explored the fundamentals of Large Language Models, figuring out how they are trained and deployed. I also started brainstorming how to combine these language models with computer vision to actually achieve the multi-modal part of this project.
