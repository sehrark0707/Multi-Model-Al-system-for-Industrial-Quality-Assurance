Week 3: Deep Learning & CNN
Summary
This week, I tackled Deep Learning fundamentals, focusing heavily on Convolutional Neural Networks for image tasks. I also explored Large Language Models, learning about tokens and transformers. Finally, I coded convolution and pooling from scratch using NumPy.

What I Got Done

Deep Learning Basics: Wrapped my head around the core components of neural networks—neurons, weights, biases, and layers. I studied why non-linear activation functions (like ReLU and Softmax) matter, and got a conceptual grasp of forward/backward propagation and gradient descent.

CNNs: Figured out exactly why we use CNNs for images instead of standard neural nets. I broke down the standard architecture (convolution layers, filters, and max pooling for downsampling) and how it all flattens out for classification. Detailed notes are saved in notes_deep_learning_cnn.md.

Intro to LLMs: Explored how text gets tokenized so models can process it. Learned about Transformer architecture and the role of self-attention, which explains how GPT-style models generate such context-aware text. Detailed notes are in notes_intro_to_llm.md.

Hands-on Practice
Instead of jumping straight into a framework, I implemented convolution and max pooling from scratch using plain NumPy. This really helped me understand what’s actually happening "under the hood" of a CNN. All the code is in cnn_concepts_demo.py.
